# deno-casbin

An authorization library that supports access control models like ACL, RBAC, ABAC in Deno. https://casbin.org/

## Installation

```typescript
import casbin from "https://deno.land/x/casbin/mod.ts";
// or directly from GitHub
import casbin from "https://raw.githubusercontent.com/casbin/deno-casbin/master/mod.ts";
```

## Usage

This library wraps the Node.js [casbin](https://github.com/casbin/node-casbin) package for use in Deno. For usage documentation, please refer to the [Node Casbin documentation](https://casbin.org/docs/get-started).
